# 第2章 Demo：Python 进阶

> 本章包含 4 个 Demo，对应 Python 进阶的四大核心知识点。
> **所有 Demo 均不需要 API Key**，可以直接运行。

---

## 学习顺序

建议按以下顺序学习：

1. **先读理论文档**：`../1.类与对象.md` → `../2.文件操作.md` → `../3.异常处理.md` → `../4.包管理与虚拟环境.md`
2. **再跑 Demo 代码**：按下面的顺序依次运行和理解

---

## Demo 列表

### Demo 1: 类与对象
**文件**: `1_demo_类与对象.py`

定义 `Document` 类和 `TextSplitter` 类，模拟 RAG 中的核心数据结构。演示类的定义、实例化、继承、方法、魔术方法。最后用这些类构建一个完整的文档处理管道。

**不需要 API Key**

---

### Demo 2: 文件操作
**文件**: `2_demo_文件操作.py`

演示读写 txt、csv、json 文件，路径处理，目录遍历。最后综合示例：扫描目录下所有文档，读取内容，统计信息，输出处理报告。

**不需要 API Key**

---

### Demo 3: 异常处理
**文件**: `3_demo_异常处理.py`

演示 try/except/finally、常见异常类型、自定义异常、重试机制。最后综合示例：模拟 RAG 管道中的异常处理，包含 API 调用重试和文件批量处理。

**不需要 API Key**

---

### Demo 4: 包管理
**文件**: `4_demo_包管理.py`

用注释和代码演示 pip、requirements.txt、venv 的用法。不依赖任何第三方库，纯 Python 代码说明包管理的概念和命令。

**不需要 API Key**

## 核心要点

学完本章后，你应该能够：

1. **用类组织代码** —— 定义 Document、Retriever 等 RAG 核心类
2. **读写各种文件** —— 加载知识库文档，保存处理结果
3. **优雅处理异常** —— API 调用失败时自动重试，文件错误时跳过继续
4. **管理项目依赖** —— 用虚拟环境隔离项目，用 requirements.txt 锁定版本
