## 1. 变量基础

### 1.1 什么是变量？

变量就是**名字标签**，贴在数据上面，方便以后找到它。

```python
name = "RAG"          # 用 name 这个标签记住 "RAG" 这串文字
score = 95.5          # 用 score 这个标签记住 95.5 这个数字
is_ready = True       # 用 is_ready 这个标签记住 True 这个状态
```

### 1.2 命名规则

- 只能包含字母、数字、下划线 `_`
- 不能以数字开头（`1name` ❌，`name1` ✅）
- 区分大小写（`Name` 和 `name` 是两个不同的变量）
- 推荐用**小写+下划线**风格：`my_variable`（snake_case）

## 4. 核心语法详解

### 4.1 切片 `[start:end]`

**❓ 你的问题：** `chunks[0:2]` 是什么意思？

**💡 解答：**

切片语法 `list[start:end]` 是**左闭右开**区间 —— 包含 start，不包含 end。

```python
chunks = ["第一段", "第二段", "第三段", "第四段"]
#          索引 0    索引 1    索引 2    索引 3

chunks[0:2]    # 取索引 0 和 1 → ["第一段", "第二段"]
chunks[1:3]    # 取索引 1 和 2 → ["第二段", "第三段"]
chunks[:3]     # 省略 start = 从头开始 → ["第一段", "第二段", "第三段"]
chunks[2:]     # 省略 end = 到末尾 → ["第三段", "第四段"]
chunks[-2:]    # 负数 = 从末尾倒数 → ["第三段", "第四段"]
```

**图示：**
```
chunks = ["第一段", "第二段", "第三段", "第四段"]
索引:       0         1         2         3
         ┌─────────┬─────────┬─────────┬─────────┐
         │  第一段  │  第二段  │  第三段  │  第四段  │
         └─────────┴─────────┴─────────┴─────────┘
            ↑                   ↑
         start=0             end=2（不包含）
         
chunks[0:2] = ["第一段", "第二段"]
```

**口诀：** `[start:end]` → 包头不包尾

### 4.2 enumerate()

**❓ 你的问题：** `for i, chunk in enumerate(chunks):` 是什么意思？

**💡 解答：**

`enumerate()` 用于在遍历时**同时获取索引和内容**。

```python
chunks = ["第一段", "第二段", "第三段"]

# 普通遍历：只能拿到内容
for chunk in chunks:
    print(chunk)
# 输出：第一段、第二段、第三段

# enumerate 遍历：同时拿到索引和内容
for i, chunk in enumerate(chunks):
    print(f"块 {i}: {chunk}")
# 输出：块 0: 第一段、块 1: 第二段、块 2: 第三段
```

**语法拆解：**
```python
for i, chunk in enumerate(chunks):
     ↑      ↑           ↑
     │      │           └── 要遍历的列表
     │      └── 每个元素的内容
     └── 每个元素的索引（从 0 开始）
```

**为什么需要索引？**
```python
# 显示"第几个"
for i, chunk in enumerate(chunks):
    print(f"[{i+1}] {chunk}")
# [1] 第一段、[2] 第二段、[3] 第三段
```

**一句话总结：** `enumerate(list)` = 遍历时同时拿到"索引"和"内容"

### 4.3 列表推导式

**❓ 你的问题：** `[len(t) for t in texts]` 是什么意思？`s for s in scores if s > 0.7` 中的 `s` 不是表达式啊？

**💡 解答：**

列表推导式是 Python 的**语法糖**，一行代码替代 for 循环。

```python
texts = ["Hello", "World", "RAG"]

# 普通 for 循环（5行）
lengths = []
for t in texts:
    lengths.append(len(t))

# 列表推导式（1行）
lengths = [len(t) for t in texts]
# 结果：[5, 5, 3]
```

**语法拆解：**
```python
[len(t) for t in texts]
   ↑        ↑      ↑
   │        │      └── 1. 从 texts 中取出每个元素
   │        └── 2. 赋值给变量 t
   └── 3. 对 t 执行操作，结果放入新列表
```

**关于 `s` 是表达式的问题：**

表达式 = **任何会产生一个值的东西**

```python
5              # 表达式 → 值是 5
len("hello")   # 表达式 → 值是 5
s              # 表达式 → 值是 s 的当前值
s * 2          # 表达式 → 值是 s 乘以 2
```

**`s` 是最简单的表达式 —— 直接返回变量本身的值**

```python
scores = [0.95, 0.3, 0.87, 0.45, 0.72]

# 情况1：对元素做变换
doubled = [s * 2 for s in scores]    # [1.9, 0.6, 1.74, 0.9, 1.44]

# 情况2：原样保留元素
high_scores = [s for s in scores if s > 0.7]    # [0.95, 0.87, 0.72]
```

**带条件的列表推导式：**
```python
scores = [0.9, 0.3, 0.8, 0.4, 0.95]
high_scores = [s for s in scores if s > 0.7]
# [0.9, 0.8, 0.95]
```

**语法总结：**
- 基本：`[表达式 for 变量 in 列表]`
- 带条件：`[表达式 for 变量 in 列表 if 条件]`

### 4.4 字典列表筛选

**❓ 你的问题：** `[r for r in search_results if r["score"] > threshold]` 是什么意思？

**💡 解答：**

这是从**字典列表**中筛选符合条件的字典。

```python
search_results = [
    {"content": "RAG...", "score": 0.95, "source": "rag_intro.pdf"},
    {"content": "向量...", "score": 0.88, "source": "vector_db.pdf"},
    {"content": "今天...", "score": 0.23, "source": "random.txt"},
]

threshold = 0.7
relevant = [r for r in search_results if r["score"] > threshold]
```

**执行过程：**
```
第 1 轮: r = {"content": "RAG...", "score": 0.95, ...}
         r["score"] = 0.95 > 0.7 → True ✓ 保留

第 2 轮: r = {"content": "向量...", "score": 0.88, ...}
         r["score"] = 0.88 > 0.7 → True ✓ 保留

第 3 轮: r = {"content": "今天...", "score": 0.23, ...}
         r["score"] = 0.23 > 0.7 → False ✗ 跳过

结果: [字典1, 字典2]
```

**等价的普通 for 循环：**
```python
# ❌ 普通 for 循环（5行）
relevant = []
for r in search_results:
    if r["score"] > threshold:
        relevant.append(r)

# ✅ 列表推导式（1行）
relevant = [r for r in search_results if r["score"] > threshold]
```

### 4.5 enumerate() 与字典列表结合

**❓ 你的问题：** `for i, result in enumerate(relevant):` 这段代码是什么意思？

**💡 解答：**

这是遍历字典列表，同时获取索引和字典。

```python
relevant = [
    {"content": "RAG...", "score": 0.95, "source": "rag_intro.pdf"},
    {"content": "向量...", "score": 0.88, "source": "vector_db.pdf"},
    {"content": "Lang...", "score": 0.91, "source": "langchain.pdf"},
]

for i, result in enumerate(relevant):
    print(f"  [{i+1}] (分数: {result['score']}) [{result['source']}] {result['content']}")
```

**执行过程：**
```
第 1 轮: i=0, result={"content": "RAG...", "score": 0.95, ...}
         输出: "  [1] (分数: 0.95) [rag_intro.pdf] RAG..."

第 2 轮: i=1, result={"content": "向量...", "score": 0.88, ...}
         输出: "  [2] (分数: 0.88) [vector_db.pdf] 向量..."

第 3 轮: i=2, result={"content": "Lang...", "score": 0.91, ...}
         输出: "  [3] (分数: 0.91) [langchain.pdf] Lang..."
```

**语法拆解：**
```python
for i, result in enumerate(relevant):
     ↑      ↑              ↑
     │      │              └── 字典列表
     │      └── 每个字典
     └── 索引（从 0 开始）
```

### 4.6 join() 方法

**❓ 你的问题：** `"\n".join([r["content"] for r in relevant])` 是什么意思？

**💡 解答：**

`join()` 用指定的分隔符把列表中的字符串**连接**起来。

```python
words = ["Hello", "World", "RAG"]

" ".join(words)          # "Hello World RAG"（用空格连接）
",".join(words)          # "Hello,World,RAG"（用逗号连接）
"\n".join(words)         # "Hello\nWorld\nRAG"（用换行连接）
```

**完整例子：**
```python
relevant = [
    {"content": "RAG...", "score": 0.95, ...},
    {"content": "向量...", "score": 0.88, ...},
    {"content": "Lang...", "score": 0.91, ...},
]

# 第 1 步：列表推导式提取 content
contents = [r["content"] for r in relevant]
# ["RAG...", "向量...", "Lang..."]

# 第 2 步：join 拼接
context = "\n".join(contents)
# "RAG...\n向量...\nLang..."

# 第 3 步：打印
print(context)
# 输出：
# RAG...
# 向量...
# Lang...
```

**在 RAG 中的作用：**
```python
# 把检索到的文档拼接成上下文，交给 LLM 生成回答
context = "\n".join([r["content"] for r in relevant])
prompt = f"请根据以下内容回答问题：\n{context}\n\n用户问题：什么是RAG？"
```

## 5. 综合示例：RAG 检索结果处理

```python
search_results = [
    {"content": "RAG通过检索外部知识来增强LLM回答", "score": 0.95, "source": "rag_intro.pdf"},
    {"content": "向量数据库用于存储Embedding向量", "score": 0.88, "source": "vector_db.pdf"},
    {"content": "今天天气真不错", "score": 0.23, "source": "random.txt"},
    {"content": "LangChain是大模型应用开发框架", "score": 0.91, "source": "langchain.pdf"},
    {"content": "Docker用于容器化部署", "score": 0.45, "source": "docker.pdf"},
]

# 筛选高分结果（条件判断 + 循环）
threshold = 0.7
relevant = [r for r in search_results if r["score"] > threshold]
print(relevant)
print(f"\n阈值: {threshold}，找到 {len(relevant)} 条相关结果:")
for i, result in enumerate(relevant):
    print(f"  [{i+1}] (分数: {result['score']}) [{result['source']}] {result['content']}")

# 拼接成上下文（构造 Prompt 的输入）
context = "\n".join([r["content"] for r in relevant])
print(f"\n拼接后的上下文:\n{context}")
```

**输出：**
```
[{'content': 'RAG...', 'score': 0.95, 'source': 'rag_intro.pdf'}, 
 {'content': '向量...', 'score': 0.88, 'source': 'vector_db.pdf'}, 
 {'content': 'Lang...', 'score': 0.91, 'source': 'langchain.pdf'}]

阈值: 0.7，找到 3 条相关结果:
  [1] (分数: 0.95) [rag_intro.pdf] RAG通过检索外部知识来增强LLM回答
  [2] (分数: 0.88) [vector_db.pdf] 向量数据库用于存储Embedding向量
  [3] (分数: 0.91) [langchain.pdf] LangChain是大模型应用开发框架

拼接后的上下文:
RAG通过检索外部知识来增强LLM回答
向量数据库用于存储Embedding向量
LangChain是大模型应用开发框架
```

**代码流程：**
1. **定义数据**：字典列表 `search_results`
2. **筛选**：列表推导式筛选 `score > 0.7` 的结果
3. **格式化输出**：`enumerate()` 遍历，f-string 格式化
4. **拼接上下文**：列表推导式提取 `content`，`join()` 拼接
