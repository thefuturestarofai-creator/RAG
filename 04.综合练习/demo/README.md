## Demo 说明

### Demo 1: 命令行待办事项管理器
**文件**: `1_demo_待办事项管理器.py`

一个完整的命令行待办事项管理器，支持：
- 添加待办事项
- 查看所有待办事项
- 标记完成
- 编辑待办事项
- 删除待办事项
- 搜索待办事项
- JSON 持久化存储

**代码中用注释标注每个部分用到了前面哪一章的什么知识点。**

**不需要 API Key**
