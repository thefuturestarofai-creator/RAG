# 第1章 Demo：Python 基础语法

> 本章包含 3 个 Demo，对应 Python 基础语法的三大核心知识点。
> **所有 Demo 均不需要 API Key**，可以直接运行。

---

## 学习顺序

建议按以下顺序学习：

1. **先读理论文档**：`../1.变量与数据类型.md` → `../2.条件判断与循环.md` → `../3.函数.md`
2. **再跑 Demo 代码**：按下面的顺序依次运行和理解

---

## Demo 列表

### Demo 1: 变量与数据类型
**文件**: `1_变量与数据类型.py`

演示 Python 的 6 大核心数据类型（str/int/float/bool/list/dict/None），以及它们在 RAG 项目中的实际用法。最后一个综合示例模拟了 RAG 检索结果的处理流程。

**不需要 API Key**

---

### Demo 2: 条件判断与循环
**文件**: `2_条件判断与循环.py`

演示 if/elif/else 条件判断、for/while 循环、break/continue 流程控制、列表推导式。综合示例构建了一个完整的"检索结果处理管道"：阈值过滤 → 排序 → 格式化 → 构造 Prompt。

**不需要 API Key**

---

### Demo 3: 函数
**文件**: `3_函数.py`

演示函数定义、参数（位置/默认/关键字/可变）、返回值、高阶函数、Lambda。综合示例用函数组织了一个完整的迷你 RAG 管道：加载 → 切分 → 检索 → 生成。

**不需要 API Key**

## 核心要点

学完本章后，你应该能够：

1. **熟练使用** list 和 dict —— RAG 项目中 80% 的数据操作都靠它们
2. **熟练使用** for 循环遍历列表 —— 遍历文档块、检索结果是 RAG 的基本操作
3. **理解** 函数的"输入-处理-输出"模式 —— RAG 管道就是多个函数的串联
4. **理解** 列表推导式 —— 简洁高效地过滤和转换数据
